# Text Classifier Project

## Instructions to Run

1. **Create and activate a virtual environment** (recommended to keep dependencies isolated):

```bash
python3 -m venv textenv
source textenv/bin/activate
```

2. **Install required Python packages**:

```bash
pip install --upgrade pip
pip install scikit-learn pandas matplotlib seaborn
```

3. **Save your script as `text_classifier.py`** in your working directory.

4. **Run the script**:

```bash
python text_classifier.py
```

5. The script will print a classification report and display a confusion matrix heatmap.
